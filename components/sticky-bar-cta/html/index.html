StickyBarCTA examples

This page demonstrates the StickyBarCTA HTML variant. The bar below is fixed to the bottom of the viewport. Click the close button to dismiss it; the dismissed state is stored in localStorage under the key in data-dismiss-key. Reload the page to see the bar stay hidden. To reset, run localStorage.removeItem("sticky-bar-dismissed") in the browser console.
