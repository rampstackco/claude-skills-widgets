# Social Proof Patterns

10-12 patterns demonstrating credibility, popularity, and existing user satisfaction.

## Patterns

| # | Name | Subcategory |
|---|---|---|
| 01 | Customer logo strip | Logo |
| 02 | Featured-in press strip | Logo |
| 03 | Single-quote testimonial | Testimonial |
| 04 | Testimonial carousel | Testimonial |
| 05 | Testimonial grid (3-6 cards) | Testimonial |
| 06 | Video testimonial | Testimonial |
| 07 | Customer count display | Quantitative |
| 08 | Real-time activity feed | Quantitative |
| 09 | Review aggregate (star rating, count) | Quantitative |
| 10 | Detailed case study card | Detailed |
| 11 | Twitter/X embed wall | UGC |
| 12 | Industry analyst quote | Authority |

## How to use this category

Match the social proof type to the audience sophistication. Logo strips work for B2B buyers who recognize the brands. Customer counts work for consumer products at scale. Detailed case studies work for considered purchases requiring proof of outcomes.

## Composes with

- `cta/` annotated CTAs incorporate social proof inline
- `urgency/` real-time activity feeds blur into both categories
- `claude-skills/skills/brand-archetype-system` archetype affects which social proof format fits (e.g. Editorial Restrained prefers single-quote testimonials; Vibrant Saturated prefers carousels)
