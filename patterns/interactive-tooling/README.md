# Interactive Tooling Patterns

8-10 patterns for interactive tools that capture user input, compute or recommend, and double as conversion surfaces.

## Patterns

| # | Name | Subcategory |
|---|---|---|
| 01 | ROI / cost calculator | Calculator |
| 02 | Savings calculator | Calculator |
| 03 | Product-market fit quiz | Quiz |
| 04 | Diagnostic quiz (assessment) | Quiz |
| 05 | Multi-step recommendation wizard | Wizard |
| 06 | Pricing tier configurator | Configurator |
| 07 | Product feature configurator | Configurator |
| 08 | Comparison tool (vs competitors) | Comparison |
| 09 | Interactive product tour | Tour |
| 10 | Scheduling tool (book a call/demo) | Scheduler |

## How to use this category

Interactive tools earn their development cost when they answer a specific question the user has before they sign up. Vanity calculators that compute irrelevant numbers fail. Tools that compute a number the user actually needs (true cost of switching, ROI on a decision, fit for their situation) drive both engagement and conversion.
